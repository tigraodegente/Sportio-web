# Sportio - A Plataforma Esportiva que Transforma Paixao em Lucro Real

> Junte-se a mais de 12.500 atletas que ja estao ganhando.
> 850 torneios/mes | 500k+ atletas | 7 esportes

## O que e o Sportio?

Sportio e a **primeira plataforma esportiva completa** que une atletas, marcas, organizadores, apostadores, fas e arbitros em um **ecossistema colaborativo** onde cada acao gera recompensas reais.

A plataforma incentiva a pratica esportiva atraves de competicoes, desafios e interacoes sociais, recompensando os usuarios com **GCoins** - a moeda digital do esporte - que podem ser convertidos em dinheiro real via PIX.

## Quick Start (Desenvolvimento)

```bash
# Clone o repositorio
git clone https://github.com/tigraodegente/Sportio-web.git
cd Sportio-web

# Instale dependencias
pnpm install

# Configure variaveis de ambiente
cp .env.example .env.local

# Execute em desenvolvimento
pnpm dev
```
